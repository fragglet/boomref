JavaScript Boom calculator

This is a JavaScript reimplementation of TRIGCALC.EXE, the command-line utility included with the Boom utilities for calculating generalized linedef types and sector types.

There are actually several different calculators for the different types:

  • Generalized floors calculator
  • Generalized ceilings calculator
  • Generalized doors calculator
  • Generalized locked doors calculator
  • Generalized lifts calculator
  • Generalized stairs calculator
  • Generalized crushing ceilings calculator
  • Generalized sector type calculator
